# frontend-template

This template contains a minimal setup of Evan's currently-favored stack for building
small (SSG) frontend projects.

Right now this is:
- pnpm
- Vite
- React
- TypeScript
- Biome 
- vscode

## Quick start

Once you have pnpm:

```
pnpm install
```

Then:

```
pnpm dev
```

Visit http://localhost:5900 to see if it works!

## Commands

To typecheck the full project:

```
pnpm tsc
```

To check linting and formatting:

```
pnpm biome
```

To fix formatting automatically:

```
pnpm format
```
